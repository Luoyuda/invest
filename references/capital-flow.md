# A 股资金流向口径

资金流向用于回答“资金是否在强化或削弱某个标的/方向”，不能单独推出买卖结论。所有资金流结论必须保留来源、统计区间、数据时间和原始口径。

## 1. 数据层级

| 资金类型 | 首选来源 | 可用补充 | 要求 |
|---|---|---|---|
| 主力资金流 | 东方财富资金流、同花顺资金流、可授权数据源 | 财联社/证券时报报道 | 必须写统计日期、净流入/流入/流出单位和订单口径 |
| 北向/沪深港通 | 交易所/港交所公开数据、沪深港通页面 | 东方财富/同花顺汇总 | 必须区分北向总额、个股持股变化、行业汇总 |
| 融资融券 | 交易所融资融券数据 | 东方财富/同花顺汇总 | 必须写交易日和余额/买入额/偿还额 |
| 龙虎榜 | 交易所龙虎榜原始披露 | 财经平台整理 | 必须写上榜日期、席位、买卖金额 |
| ETF/产业资本 | 基金公告、交易所、上市公司公告 | 主流财经媒体 | 必须写份额/金额/公告时间 |

## 2. 字段口径

资金流最终应尽量结构化为：

```json
{
  "code": "300308",
  "name": "中际旭创",
  "provider": "eastmoney",
  "data_time": "2026-05-15",
  "latest": {
    "main_net_inflow_yi": -7.113,
    "main_net_inflow_pct": -2.1,
    "breakdown_yi": {
      "super_large": -0.788,
      "large": -0.772,
      "medium": -0.366,
      "small": 1.927
    }
  },
  "recent": {
    "3d_main_net_inflow_yi": -16.4822,
    "5d_main_net_inflow_yi": -23.2037,
    "10d_main_net_inflow_yi": -30.6511,
    "20d_main_net_inflow_yi": -51.3
  }
}
```

说明：

- `main_net_inflow_yi` 是主力净流入，单位亿元。
- `breakdown_yi` 是超大单、大单、中单、小单净流入，单位亿元。
- `realtime.total_inflow_yi` / `realtime.total_outflow_yi` 只有在来源明确提供流入/流出总额时才可写。
- 如果来源只提供净流入，不得反推“主力流入”和“主力流出”总额。
- 如果截图或外部页面提供流入/流出总额，必须标注该页面来源和统计日期。

## 2.1 本仓库资金流 Provider

| Provider | 用途 | 稳定性 | 说明 |
|---|---|---|---|
| `eastmoney` | 个股 3/5/10/20 日主力净流入趋势 | S3 | 默认趋势源；提供净流入和超大/大/中/小单净额 |
| `ths` | 同花顺个股实时资金流入/流出和大/中/小单拆分 | S4 | 可选增强源；公开网页服务接口，可能受 cookie、Referer、反爬影响 |
| `auto` | 东方财富趋势 + 同花顺实时字段 | S3/S4 | 默认模式；同花顺失败时保留东方财富趋势并降级 |
| `ths --scope market` | 同花顺沪深两市个股资金流向排行样本 | S4 | 用于观察大盘资金榜方向；样本合计不得冒充全市场精确总额 |

脚本入口：

```bash
python3 scripts/fetch_capital_flow.py 300308 --provider auto --days 20
python3 scripts/fetch_capital_flow.py --scope market --provider ths --limit 50
```

## 3. 解读规则

| 现象 | 可以说 | 不可以说 |
|---|---|---|
| 当日主力净流入为正 | 当日该来源口径下主力资金净流入 | 主力一定在建仓 |
| 3/5/10/20 日持续净流出 | 近期资金面偏弱或筹码承压 | 后面一定下跌 |
| 小单净流入、主力净流出 | 散户承接或资金结构分化，需要结合价格位置验证 | 主力出货已经确定 |
| 主力净流入但股价下跌 | 资金与价格信号背离，需要看成交额和板块 | 低吸机会 |
| 主力净流出但股价上涨 | 可能是资金分歧或统计口径差异，需要交叉验证 | 拉高出货确定 |

## 4. 推荐链路使用

- 资金流只能作为推荐的辅助证据，不得替代行业/政策/产业催化、基本面和价格位置。
- 推荐前 5 的标的如果资金流为明显连续净流出，必须降低确定性或写入风险。
- 热门板块如果只有价格热度、没有资金验证，必须标注“资金面未核验”。
- 资金流端点失败时，不得补编“资金流入/流出”；应写“资金面未取得可靠结构化数据”。

## 5. 质量清单

- 是否写明资金来源和统计日期。
- 是否区分当日、3 日、5 日、10 日、20 日。
- 是否保留原始净流入字段和单位换算。
- 是否避免把净流入反推成完整流入/流出总额。
- 同花顺实时字段失败时是否降级，而不是阻断定时任务。
- 是否说明资金流不能单独构成买卖建议。
