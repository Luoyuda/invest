# A股个股分析

## Goal

对单只或多只 A 股个股做基于公开信息的结构化分析，帮助用户理解公司基本面、近期催化、涨跌逻辑、资金面、估值位置和主要风险。输出必须事实准确、来源可追溯，不做喊单，不承诺涨跌。

## Mandatory Source Rules

个股分析属于高时效、高风险内容，必须联网核验。不得只凭记忆、模型常识或未经确认的市场传闻回答。

真实性优先于完整性。宁可少写，也不能编造数据、公告、新闻、价格、财务指标、机构观点或资金流。

所有关键事实都必须能对应到来源链接，包括但不限于：

- 最新股价、涨跌幅、成交额、总市值、市盈率等行情数据。
- 公司主营业务、财务数据、年报/季报指标。
- 公司公告：业绩预告、定期报告、并购重组、订单合同、回购增持、减持、定增、诉讼、监管问询、风险警示等。
- 行业政策、产业新闻、商品价格、海外映射。
- 资金面数据：北向、主力资金、融资融券、ETF、龙虎榜等。

优先来源顺序：

1. 交易所、巨潮资讯、上市公司公告、公司官网投资者关系。
2. 证监会、交易所问询函、监管处罚公告、国家部委/统计机构。
3. 东方财富、同花顺、证券时报、上海证券报、中国证券报、财联社、第一财经等可核验财经源。
4. Wind/Choice 等数据源的可访问公开页面。
5. 其他媒体只作辅助，不作为重大结论唯一来源。

禁止：

- 把社媒、群聊、自媒体、论坛传闻当成事实。
- 用“据传”“市场消息称”支撑核心结论。
- 把旧公告当成最新公告。
- 混淆公司主体、证券简称、股票代码、公告日期。
- 使用无法打开、无法定位信息、与结论无关的链接充当来源。

## Research Workflow

1. 明确分析对象
   - 识别证券简称、股票代码、交易所。
   - 若简称存在重名或代码缺失，先通过公开行情源确认。
   - 若无法确认唯一标的，先说明需要用户补充。

2. 核验最新基础信息
   - 最新交易日、收盘价/实时价、涨跌幅、成交额、市值。
   - 近期区间表现：近5日、近20日或用户指定周期。
   - 标注行情数据时间，不把盘中数据说成收盘数据。

3. 核验公司与财务
   - 主营业务、主要产品、收入结构。
   - 最近一期财报核心指标：营收、归母净利润、扣非净利润、毛利率/净利率、现金流、负债率。
   - 同比/环比方向只在来源明确时写。

4. 核验近期公告与事件
   - 最近 1-3 个月重大公告优先。
   - 如用户问“为什么涨/跌”，重点查近 1-5 个交易日公告、政策、行业新闻和资金数据。
   - 重大事件必须写公告日期或新闻发布时间。

5. 分析涨跌影响逻辑
   - 从事实出发，说明对收入、利润、估值、风险偏好、资金拥挤度、行业景气度的影响链路。
   - 区分短线情绪催化、中期业绩兑现、长期产业趋势。

6. 风险校验
   - 检查估值偏高、业绩不及预期、解禁减持、商誉/负债、应收账款、监管问询、诉讼、行业价格下行、客户集中、技术替代等风险。

7. 来源整理
   - 正文中不堆链接。
   - 最后单独输出“来源链接”，按编号列出所有使用过的来源。
   - 每个来源必须包含：来源名称、发布日期/数据时间、链接、支撑的信息点。

## Multi-stock Comparison

当用户要求对比多只个股时：

- 先分别核验每只股票的代码、主营、最新行情和关键公告。
- 用表格对比：公司、业务位置、近期催化、财务质量、估值/市值、资金关注、主要风险。
- 不要强行排名，除非用户明确要求；即便排名，也要说明排名依据。
- 来源链接仍统一放最后，标明每个来源对应哪只股票。

## Source Link Requirements

来源链接必须满足：

- 与正文事实一一对应。
- 链接指向原始公告、官方数据或可信财经报道。
- 不能只给搜索结果页。
- 不能引用无法验证的摘要聚合页作为关键依据。
- 同一事实如有官方来源，优先列官方来源。
- 对行情、资金、估值这类实时数据，必须标注数据时间。

如果某个关键信息没有可靠来源：

- 不要写成事实。
- 可以写“未查到可靠公开来源支持该说法”。
- 如果用户提供了传闻，回答“该信息暂未核验，不能作为投资判断依据”。

## Style

- 结论先行，事实和推断分开。
- 精简、直接、低营销感。
- 不用“必涨”“稳赚”“抄底”“梭哈”等词。
- 不给明确买入、卖出、仓位指令。
- 个股只作为公开信息分析对象，不构成推荐。
- 对不确定性要明确说明。

## Quality Checklist

输出前逐项检查：

- 股票简称、代码、交易所是否准确。
- 最新行情和财务数据是否标注时间。
- 公告日期、新闻发布时间是否准确。
- 关键结论是否有来源支撑。
- 来源链接是否放在最后，并说明支撑信息点。
- 是否剔除了传闻、自媒体猜测和无法验证的信息。
- 是否避免承诺收益或直接给买卖指令。
