# Skill 索引

## 1. A 股核心财经要闻

路径：

```text
skills/a-share-market-news/SKILL.md
```

用途：

- 整理指定日期、近 1 日、近 3 日、近一周影响 A 股盘面和个股的核心财经要闻。
- 按利好/利空分类。
- 覆盖宏观政策、行业政策、经济数据、上市公司公告、资金动向、外围市场和大宗商品。

触发词：

- A 股要闻
- 利好利空
- 盘前/盘后
- 近 1 日 / 近 3 日 / 近一周
- 上周最后一个交易日

## 2. A 股个股分析

路径：

```text
skills/a-share-stock-analysis/SKILL.md
```

用途：

- 分析单只或多只 A 股个股。
- 覆盖行情、基本面、财务、估值、公告、资金、风险。
- 所有来源链接附在最后。

触发词：

- 分析某股票
- 某股票怎么看
- 为什么涨/跌
- 公告影响
- 基本面/估值/资金面

## 3. A 股个股候选推荐

路径：

```text
skills/a-share-stock-recommendation/SKILL.md
```

用途：

- 推荐 A 股候选观察标的。
- 按稳健价值、成长趋势、高股息防守、事件催化等框架筛选。
- 输出推荐理由、风险、逻辑失效条件、来源链接。

触发词：

- 推荐股票
- 找潜在机会
- 筛选股票
- 候选观察池
- 哪些股票值得关注

硬边界：

- 推荐只表示候选观察，不是买入指令。
- 不给目标价、止损价、仓位比例。
